### Frontend simples inspirado no funcionamento do Instagram

#### Features

- Criação de posts
- listagem de posts
- função de like nas fotos
- double tap to like

#### Stack utilizada

- React.js
- React router
- Styled components
- Redux
- Redux-saga
- axios
