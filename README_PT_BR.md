# Frontend simples inspirado no funcionamento do Instagram

#### :page_facing_up: Features

- criação de posts
- listagem de posts
- like nos posts com double tap to like

#### :computer: ​Stack utilizada

- React.js
- React router
- Styled components
- Redux
- Redux-saga
- axios
